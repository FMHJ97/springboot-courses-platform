# Spring Boot — Courses Platform API

Practice project REST API with authentication, authorization and database persistence.

## Technologies
- Java 21 · Spring Boot · Spring Security · JWT · PostgreSQL · JPA · Maven

## Features
- JWT authentication (register / login)
- Role-based access control (ADMIN / INSTRUCTOR / STUDENT)
- REST API with CRUD endpoints
- Persistence with JPA/Hibernate and PostgreSQL
- Global exception handling (`@RestControllerAdvice`)
- DTO pattern
- API documentation with OpenAPI / Swagger

## In progress
- Unit tests with JUnit 5 and Mockito
- Remaining CRUD endpoints (Course, Lesson, Enrollment)
